# 魔方速拧训练器

一个纯前端、本地运行的魔方速练工具。使用 Solid.js + Vite 构建，数据全部存储在浏览器 IndexedDB 中，无需服务端。

## 功能特性

- ⏱️ **精准计时器**：长按空格开始、抬手停止，支持 +2 / DNF 罚时
- 📊 **成绩统计**：自动计算单次最好、五次平均（去头尾）、十二次平均等
- 📈 **趋势图表**：每日成绩曲线可视化，直观看到进步
- 📝 **公式库管理**：按层先、CFOP 等分组管理公式，支持导入导出
- 💾 **本地存储**：所有数据保存在 IndexedDB，不上传服务器
- 📱 **触屏支持**：移动端也能长按屏幕计时

## 本地启动

### 环境要求

- Node.js >= 16
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

启动后访问 http://localhost:5173

### 生产构建

```bash
npm run build
```

构建产物输出到 `dist/` 目录，可以直接部署到任意静态服务器。

### 预览构建产物

```bash
npm run preview
```

## 示例公式导入

项目内置了一套示例公式（层先法、CFOP 基础等），可以一键导入：

1. 打开应用，点击顶部「公式库」标签页
2. 点击右上角「导入示例」按钮
3. 确认后即可看到预置的 4 个分组和 12 条公式

### 手动导入 JSON

你也可以导入自己的公式库 JSON 文件，格式如下：

```json
{
  "formulas": [
    {
      "id": "f-001",
      "name": "顶层十字",
      "notation": "F R U R' U' F'",
      "description": "OLL 基础公式",
      "createdAt": 1700000000000
    }
  ],
  "groups": [
    {
      "id": "g-001",
      "name": "层先法",
      "description": "初学者入门的层先法公式",
      "formulaIds": ["f-001"],
      "createdAt": 1700000000000
    }
  ]
}
```

点击「导入」按钮选择 JSON 文件即可。

### 导出公式

点击「导出」按钮可以将当前公式库导出为 JSON 文件，方便备份或分享。

## 架构说明

### 分层设计

项目按职责分为四层，各层之间单向依赖：

```
UI 组件 → 状态管理 → 数据层 → IndexedDB
         ↘ 工具函数 ↙
```

- **工具层 (`utils/`)**：纯函数，无副作用，不依赖 Solid.js 或浏览器 API（除了 chart 依赖 Canvas）
- **数据层 (`db/`)**：IndexedDB 封装，只做数据存取，不包含业务逻辑
- **状态层 (`stores/`)**：业务状态管理，组合工具函数和数据层
- **组件层 (`components/`)**：UI 渲染和用户交互，只调用 store 的方法

这种分层方式的好处：
- 计时逻辑、统计算法、公式管理完全解耦
- 图表绘制独立为纯 Canvas 模块，不依赖组件框架
- 以后加「分步还原教学」等新功能时，可以复用工具和数据层
- 单元测试可以只测纯函数，不需要模拟浏览器环境

### 核心算法

**五次去头尾平均 (ao5)**：取最近 5 次成绩，去掉最快和最慢各一次，剩下 3 次取平均。如果有 2 次及以上 DNF，则平均为 DNF。

**每日统计**：按自然日分组，计算每天的最好成绩、五次平均、平均成绩。

## 使用技巧

- 长按空格键约 0.3 秒后变绿，松开即开始计时
- 可以给记录标记 +2 或 DNF，统计时会自动计算有效成绩
- 练特定公式组时可以在公式库选中分组，计时会自动关联
- 成绩数据都在本地，清除浏览器数据会丢失，记得定期导出备份
